# Fitness-Managetment-and-CRM

A full-stack web application for managing fitness center memberships, tracking member Buying, and supporting staff with a centralized CRM dashboard.

---

## Features

- Member registration and subscription management
- CRM system to track member activity and visit history
- Staff dashboard for managing member data and improving operational efficiency
- Responsive design for desktop and mobile use
- REST API backend with role-based data access

---

## Tech Stack

**Frontend**
- React
- Tailwind CSS
- JavaScript

**Backend**
- REST API

**Database**
- MySQL

---

## Project Structure

```
├── fitness/          # React frontend
└── backend/          # Laravel backend & REST API
```

---

## Getting Started

### Prerequisites

- Node.js >= 18
- MySQL

### Installation

**1. Clone the repository**
```bash
git clone https://github.com/tonywn7895/Fitness-Management-and-CRM.git
cd Fitness-Management-and-CRM
```

**2. Setup backend**
```bash
cd backend
composer install
cp .env.example .env
```



**3. Setup frontend**
```bash
cd fitness
npm install
npm run dev
```
